# Source package for Encode/Decode Packet Data Unit (PDU) (7bit, UCS2)
> - If your app using GSM SMS functionality
> - Asterisk
> - Some tests

### Functionality

- Decode PDU to normal texts
- Encode normal texts to PDU

### Add to use

```
go get github.com/KusoKaihatsuSha/gopdu.git
```
